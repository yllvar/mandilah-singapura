# Jupyter Notebooks

This folder contains interactive Jupyter notebooks for exploratory data analysis and visualization.

## Notebooks

- `01_exploratory_analysis.ipynb` - Initial data exploration and visualization
- `02_regime_analysis.ipynb` - Deep dive into different time regimes
- `03_strategy_development.ipynb` - Strategy development and testing
- `04_visualization_gallery.ipynb` - Collection of plots and charts

## Usage

```bash
# Start Jupyter notebook server
jupyter notebook notebooks/

# Or use JupyterLab
jupyter lab notebooks/
```

## Dependencies

Notebooks require the same dependencies as the main scripts, plus:
- jupyter
- ipywidgets (for interactive widgets)

Install with:
```bash
pip install jupyter ipywidgets
```

## Purpose

These notebooks are designed for:
- Interactive exploration of the MYR/SGD relationship
- Experimenting with different analysis approaches
- Creating visualizations for presentations
- Documenting analysis workflows
